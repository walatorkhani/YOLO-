YOLO Real Time Object Detection:

This project uses YOLOv8 with OpenCV to detect objects in real time using a webcam.

Description:

The system captures video from a webcam and applies a pre-trained YOLOv8 model to detect objects in each frame.
Each detection includes:
Object class
Bounding box location
Confidence score

Technologies Used:
Python
YOLOv8 (Ultralytics)
OpenCV
